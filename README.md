# Statistics For NSW HSC Science Extension

The intended audience for these notes is students of the NSW HSC Science Extension (SCIX) course, although they would be a suitable introduction to statistics in science for anyone who is approaching these concepts for the first time. This is not a course in mathematical statistics, so concepts such as probability distributions are approached through examples rather than theory. Only minimal experience with spreadsheets is required, and all statistical analysis can be completed using the recommended online resources.

I hope you find these notes a useful map for your statistical journey.
